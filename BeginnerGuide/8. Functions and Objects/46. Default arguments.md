어떤 프로그래밍 언어에서든 함수는 매우 강력한 도구입니다. 때로는 함수의 모든 인수를 사용할 필요가 없습니다. 일부 인수는 미리 선택된 값으로 저장되어 있을 수 있기 때문입니다. 이 글에서는 그 방법을 알아보겠습니다.

## 기본 인수가 있는 함수

Kotlin은 함수 선언에서 [함수 매개변수](https://hyperskill.org/learn/step/4637 "In Kotlin, a function parameter is a variable that is declared in the function header and provides input to the function to perform a specific task. | These parameters can be constants or variables, and they can also be other named arguments or functions. When invoking a function with parameters, we can pass arguments by the name of these parameters to improve code readability, especially if a function has many parameters. Additionally, this approach is useful when changing the order of arguments passed in a function call. For example, when calculating the total money a cashier will have at the end of the workday, a function can be created with three parameters: the initial sum in the cash desk, the price of a ticket, and the number of tickets sold that day.") 에 **[기본값을](https://hyperskill.org/learn/step/4637 "Kotlin에서 기본값은 함수 매개변수에 미리 할당된 값입니다. 즉, 해당 매개변수에 값을 명시적으로 지정하지 않고도 함수를 호출할 수 있으며, 기본값이 대신 사용됩니다. | 이 기능을 사용하면 모든 인수를 지정할 필요가 없는 경우 더욱 간결한 함수 호출이 가능합니다. 기본값은 함수 선언에서 타입 뒤에 `=` 연산자를 사용하여 지정되며, 함수를 호출할 때 명명된 인수를 제공하여 재정의할 수 있습니다.")** [할당할 수 있습니다. 함수를 호출](https://hyperskill.org/learn/step/4637 "In Kotlin, invoke refers to the action of calling or using a function. | When we want to use a function, we can invoke it by using its name followed by parentheses. If a function takes one or more arguments (input data), they should be passed inside the parentheses. For example, if a function takes a single-string argument, it can be invoked like this: `functionName(string)`. Some functions not only take arguments but also produce a result, which can be assigned to a variable. Function arguments and producing a result are fundamental concepts when working with functions in Kotlin.") 하려면 기본값이 있는 인수를 생략하거나 일반적인 방식으로 호출할 수 있습니다.[](https://hyperskill.org/learn/step/4637 "Kotlin에서 함수 매개변수는 함수 헤더에 선언되어 특정 작업을 수행하기 위해 함수에 입력을 제공하는 변수입니다. | 이러한 매개변수는 상수 또는 변수일 수 있으며, 다른 명명된 인수 또는 함수일 수도 있습니다. 매개변수가 있는 함수를 호출할 때, 특히 함수에 매개변수가 많은 경우 코드 가독성을 높이기 위해 이러한 매개변수의 이름으로 인수를 전달할 수 있습니다. 또한, 이 방법은 함수 호출에서 전달되는 인수의 순서를 변경할 때 유용합니다. 예를 들어, 계산원이 근무일 종료 시 받게 될 총액을 계산할 때, 계산대의 초기 금액, 티켓 가격, 그리고 그날 판매된 티켓 수라는 세 가지 매개변수를 갖는 함수를 만들 수 있습니다.")[](https://hyperskill.org/learn/step/4637 "코틀린에서 'invoke'는 함수를 호출하거나 사용하는 행위를 의미합니다. | 함수를 사용하려면 함수 이름 뒤에 괄호를 붙여 호출할 수 있습니다. 함수가 하나 이상의 인수(입력 데이터)를 받는 경우, 괄호 안에 전달해야 합니다. 예를 들어, 함수가 단일 문자열 인수를 받는 경우 `functionName(string)`과 같이 호출할 수 있습니다. 일부 함수는 인수를 받을 뿐만 아니라 변수에 할당할 수 있는 결과를 생성하기도 합니다. 함수 인수와 결과 생성은 코틀린에서 함수를 사용할 때 기본적인 개념입니다.")

다음은 두 개의 매개변수를 갖는 함수입니다 `printLine`. 첫 번째 매개변수는 줄을 정의하고, 두 번째 매개변수는 이 줄을 끝내는 문자열입니다. 두 매개변수 모두 기본값을 갖습니다. 빈 줄( `""`)과 새 줄( `"\n"`)입니다.

```kotlin
fun printLine(line: String = "", end: String = "\n") = print("$line$end")
```

이 예를 살펴보세요. `=`유형 뒤에 연산자를 사용하여 매개변수에 값을 할당합니다.

전체 코드 조각을 살펴보세요. 다음과 같은 기능이 있습니다. `printLine`및 `main`.

```kotlin
fun printLine(line: String = "", end: String = "\n") = print("$line$end")

fun main() {
    printLine("Hello, Kotlin", "!!!") // prints "Hello, Kotlin!!!"
    printLine("Kotlin") // prints "Kotlin" with an ending newline
    printLine() // prints an empty line like println()
}
```

`"Hello, Kotlin"`첫 번째 호출에서는 두 개의 인수( 와 ) 가 함수에 전달됩니다 `"!!!"`. 두 번째 호출에서는 하나의 인수( `"Kotlin"`)만 전달되고, 두 번째 매개변수의 값은 기본값( `"\n"`)입니다. 세 번째 호출에서는 인수를 전달하지 않지만, 두 매개변수 모두 기본값을 가지고 있기 때문에 여전히 작동합니다.

프로그램 출력:

```kotlin
Hello, Kotlin!!!Kotlin 
```

첫 번째 인수 없이 두 번째 인수를 전달할 수 없습니다. Kotlin은 두 번째 매개변수에만 값을 할당하려는 것을 이해하지 못합니다.

## 기본 인수와 일반 인수 혼합

선언 시 기본 매개변수와 일반 매개변수를 혼용할 수도 있습니다. 예를 들어, 아래 함수는 두 정수 인수의 최댓값을 구합니다. 이 함수에는 숫자의 절댓값을 비교할지 여부를 결정하는 특수 매개변수가 있습니다. 이 매개변수의 기본값은 입니다 `false`.

```kotlin
fun findMax(n1: Int, n2: Int, absolute: Boolean = false): Int {
    val v1: Int
    val v2: Int

    if (absolute) {
        v1 = Math.abs(n1)
        v2 = Math.abs(n2)
    } else {
        v1 = n1
        v2 = n2
    }

    return if (v1 > v2) n1 else n2
}

fun main() {
    println(findMax(11, 15)) // 15
    println(findMax(11, 15, true)) // 15
    println(findMax(-4, -9)) // -4
    println(findMax(-4, -9, true)) // -9
}
```

함수 본문에서 `findMax`두 개의 임시 변수를 선언합니다. 이 변수들은 absolute 값에 따라 절대값 또는 동일한 인수 값을 저장합니다. 마지막 줄에서는 임시 변수를 비교하여 해당 인수의 값을 반환합니다.

`findMax`함수 본문에서 네 번 호출합니다 `main`. 첫 번째 호출은 최댓값( `15`)을 반환하고, 두 번째 호출은 양수의 최댓값( `15`)을 반환하며, 세 번째 호출은 두 음수의 최댓값( `-4`)을 반환하고, 마지막 호출은 두 음수의 절댓값으로 최댓값을 반환합니다.

[기본 인수를](https://hyperskill.org/learn/step/4637 "코틀린에서 기본 인수는 함수 호출 시 인수가 전달되지 않으면 기본값을 받는 함수의 매개변수입니다. | 기본 인수를 사용하면 기본값을 가진 모든 매개변수에 인수를 제공할 필요가 없으므로 함수 호출을 더 간결하게 만들 수 있습니다. 그러나 기본 인수와 일반 인수를 함께 사용할 경우 어떤 매개변수에 인수를 받아야 할지 모호할 수 있습니다. 혼란을 피하기 위해 명명된 인수를 기본 인수와 함께 사용할 수 있습니다. 기본값은 상수이거나 명명된 인수 또는 함수일 수 있지만, 반드시 초기화해야 합니다.") 사용하면 프로그래머가 함수에 대한 **선택적** 또는 **특수한** 동작을 만들 수 있습니다 .
