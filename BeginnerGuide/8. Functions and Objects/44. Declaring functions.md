코틀린에서 함수는 코드의 기반을 형성합니다. 함수는 프로그램 전체에서 실행하고 재사용할 수 있는 코드 블록을 정의합니다. 함수는 작동을 위해 구성된 일련의 명령어입니다. 어떤 면에서는 일종의 하위 프로그램입니다.

함수는 코드를 관리하기 쉬운 단위로 정리하기 때문에 중요한 역할을 합니다. 코드 재사용을 가능하게 하고, 중복을 줄이며, 가독성을 향상시킵니다. 특정 작업을 함수 안에 래핑하면 더욱 모듈화되고 유지 관리하기 쉬운 코드베이스를 구축할 수 있습니다.

새 함수를 선언하려면 키워드와 함수 이름, 괄호 안의 매개변수 목록, 그리고 결과 값의 타입을 명시해야 합니다 `fun`. 다음과 같습니다.

매개변수의 유형과 결과는 `Int`, `Double`, `Boolean`, `String`등을 포함한 모든 유형이 될 수 있습니다.

## 간단한 함수 정의

정수의 합을 계산하고 이를 반환하는 함수를 선언해 보겠습니다.

```kotlin
fun sum(a: Int, b: Int): Int {
    val result = a + b
    return result
}

fun main() {
    val result1 = sum(2, 5)
    println(result1) // 7
    
    val result2 = sum(result1, 4)
    println(result2) // 11
}
```

함수를 처음 호출할 때 두 개의 값 `2`과 를 전달합니다. 이 값들을 각각 와 매개변수 `5`에 할당합니다 . 함수는 결과 값을 반환합니다. 반환 타입은 함수( )에 선언된 반환 값의 타입과 동일합니다 . 두 번째 호출할 때는 와 의 값을 인수로 전달합니다 . 함수는 를 반환 합니다 .`a``b``result``Int``result1``7``4``11`

함수의 인수는 함수 내에서만 접근할 수 있습니다.

## 함수 매개변수

아시다시피 매개변수는 함수의 입력 데이터를 나타냅니다. 동일하거나 다른 유형의 매개변수를 하나 이상 받는 함수를 선언할 수 있습니다. 또한, 매개변수가 전혀 없더라도 괄호는 여전히 필요한 함수를 선언할 수 있습니다.

다음은 몇 가지 예입니다.

```kotlin
/**
 * The function returns its argument
 */
fun identity(a: Int): Int {
    return a
}

/**
 * The function returns the sum of two Ints
 */
fun sum(a: Int, b: Int): Int {
    return a + b
}

/**
 * The function just returns 3
 */
fun get3(): Int {
    return 3
}

fun add3(a: Int): Int {
    return identity(a) + get3()
}

fun main() {
    println(identity(1000)) // 1000
    println(sum(200, 300))  // 500    
    println(get3())         // 3
    println(add3(5))        // 8
}
```

보시다시피, `main`함수에서 함수를 호출할 수 있지만, function 처럼 다른 일반 함수에서 함수를 호출하는 것도 가능합니다 `add3()`.

## 반환 유형

함수는 단일 값을 반환하거나 아무것도 반환하지 않을 수 있습니다. 함수가 무언가를 반환하는 경우, 함수 본문에는 `return`키워드 다음에 결과가 나와야 합니다. 때로는 아무것도 반환하지 않는 함수가 필요할 수 있습니다. 아무것도 반환하지 않는 함수를 선언하는 두 가지 방법이 있습니다.

- [반환 유형](https://hyperskill.org/learn/step/4580 "코틀린에서 반환 타입은 함수가 반환하는 값의 타입입니다. | 함수가 값을 반환하지 않으면 반환 타입은 `Unit`으로 표시되며, 이는 코틀린에서 알려진 모든 타입의 하위 타입입니다. 이는 분모가 0인 백분율을 계산하는 경우처럼 함수가 값을 반환할 수 없는 상황에서 유용합니다. 이러한 경우 `Nothing`을 반환하면 함수가 null이나 다른 임의의 값을 반환하는 대신 값을 반환하지 않음을 나타냅니다.") 부분을 생략합니다 .
    

```kotlin
/**
 * The function prints the values of a and b
 */
fun printAB(a: Int, b: Int) {
    println(a)
    println(b)
}
```

- `Unit`반환 유형으로 특수 유형을 지정합니다 .
    

```kotlin
/**
 * The function prints the sum of a and b
 */
fun printSum(a: Int, b: Int): Unit {
    println(a + b)
}
```

## 함수 본문

[함수 본문](https://hyperskill.org/learn/step/4580 "코틀린에서 함수 본문은 함수가 수행해야 할 작업을 정의하는 프로그래밍 문장을 포함하는 함수의 한 부분입니다. | 중괄호 `{}`로 묶이며, 0개 이상의 문장을 포함할 수 있습니다. 예를 들어, hello world 프로그램의 함수 본문에는 문자열 hello, world!를 출력하고 그 뒤에 줄 바꿈을 하는 문장이 포함됩니다. 함수 본문이 비어 있으면 함수가 호출될 때 아무런 동작이나 계산도 수행되지 않습니다.") (예: 함수 내부) 에 원하는 문장을 작성할 수 있습니다 `main`. 매개변수 외에도 함수 내에서만 볼 수 있는 새 변수를 선언할 수 있습니다.

예를 들어, 이 함수는 숫자의 마지막 숫자를 추출한 다음 반환합니다.

```kotlin
fun extractLastDigit(number: Int): Int {
    val lastDigit = number % 10
    return lastDigit
}
```

추가 변수를 제외하면 이 코드를 단순화할 수 있습니다.

```kotlin
fun extractLastDigit(number: Int): Int {
    return number % 10
}
```

함수 본문에서 모든 연산을 수행할 수 있습니다. 예를 들어, 이 함수는 관계 연산을 사용합니다. 숫자가 양수인지 아닌지 확인합니다.

```kotlin
fun isPositive(number: Int): Boolean {
    return number > 0
}
```

이 함수는 `isPositive`정수를 입력받아 `Boolean`표현식의 결과가 또는 이기 때문에 값을 `number > 0`반환 `true`합니다 `false`.

한 가지 예를 더 살펴보겠습니다. return 문 다음 줄은 실행되지 않습니다( **도달할 수 없는** 코드라고 함).

```kotlin
/**
 * It returns "hello"
 */
fun getGreeting(): String {
    return "hello"   // Ends the function
    println("hello") // Will not be executed
}
```

## 단일 표현식 함수

[함수가 단일 표현식을 반환하는 경우 중괄호](https://hyperskill.org/learn/step/4580 "코틀린에서 중괄호 `{}`는 0개 이상의 명령문을 포함할 수 있는 코드 블록을 나타내는 데 사용됩니다. | 또한 문자열 템플릿에서 복잡한 표현식이나 객체 속성을 평가하는 데에도 중괄호를 사용합니다. 표현식이 더 복잡한 경우, 달러 기호 `$` 다음에 중괄호를 사용하여 표현식을 감싸고 사용할 수 있습니다. 문자열 템플릿에서는 실수를 방지하고 코드 가독성을 높이기 위해 항상 표현식에 중괄호를 사용하는 것이 좋습니다.") 없이 작성할 수 있습니다 . 이는 간결한 함수를 만드는 관용적인 방법입니다.

```kotlin
fun sum(a: Int, b: Int): Int = a + b

fun sayHello(): Unit = println("Hello")

fun isPositive(number: Int): Boolean = number > 0
```

반환 유형을 지정하는 것은 선택 사항이며 자동으로 추론될 수 있습니다.

```kotlin
fun sum(a: Int, b: Int) = a + b // Int

fun sayHello() = println("Hello") // Unit

fun isPositive(number: Int) = number > 0 // Boolean
```
