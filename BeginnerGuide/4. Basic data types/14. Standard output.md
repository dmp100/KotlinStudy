**표준 출력은** 장치에 정보를 표시하는 기본적인 동작입니다. 모든 프로그램이 이러한 출력을 생성하는 것은 아닙니다. 기본적으로 표준 출력은 데이터를 화면에 표시하지만, 파일로 리디렉션할 수도 있습니다.

Hyperskill에서는 문자열이나 숫자 등의 데이터를 표준 출력으로 보내는 프로그램을 자주 작성하게 됩니다.

## 텍스트 인쇄

Kotlin에는 표준 출력으로 데이터를 전송하는 두 가지 함수가 있습니다: `println`및 `print`.

`println`( **print line** ) 함수는 문자열을 화면에 표시한 후 줄 바꿈을 합니다 . **예** 를 들어, 아래 코드 조각은 네 줄을 출력 합니다 **.**

```kotlin
println("I")
println("know")
println("Kotlin")
println("well.")
```

**산출:**

```java
I
know
Kotlin
well.
```

보시다시피, 모든 문자열은 [큰따옴표](https://hyperskill.org/learn/step/4425 "코틀린에서 큰따옴표는 일련의 문자를 나타내는 문자열 유형을 정의하는 데 사용되는 기호입니다. | 코틀린에서 가장 널리 사용되는 데이터 유형 중 하나이며, 큰따옴표로 묶인 문자 수에 따라 결정됩니다. 문자열 연결은 문자열에서 흔히 사용되는 연산으로, 다른 문자열들을 사용하여 하나의 문자열을 구성하는 데 사용됩니다. 두 문자열을 연결하면 새로운 문자열이 생성됩니다.") 없이 인쇄됩니다 .

빈 줄을 인쇄할 수도 있습니다.

```kotlin
println("Kotlin is a modern programming language.")
println() // prints an empty line
println("It is used all over the world!")
```

**산출:**

```java
Kotlin is a modern programming language.

It is used all over the world!
```

이 함수는 **값을**`print` 표시 하고 커서를 **뒤에** 놓습니다 . 아래 예를 살펴보겠습니다. 이 코드는 모든 문자열을 한 줄에 출력합니다.

```kotlin
print("I ")
print("know ")
print("Kotlin ")
print("well.")
```

**산출:**

```java
I know Kotlin well.
```

## 숫자와 문자 인쇄

`println`및 함수를 사용하면 `print`프로그램은 문자열뿐만 아니라 **숫자** 와 **문자** 도 출력할 수 있습니다 .

두 개의 비밀 코드를 출력해 보겠습니다.

```kotlin
print(108)   // prints a number
print('c')   // prints a character
print("Q")   // prints a string
println('3') // prints a character that represents a digit 

print(22)
print('E')
print(8)
println('1')
```

**산출:**

```java
108cQ3
22E81
```

문자열과 마찬가지로 따옴표는 사용할 수 없습니다.

## 운영자`$`​

Kotlin 프로그래밍 언어에서 `$`연산자는 종종 [문자열 템플릿 에서 변수나](https://hyperskill.org/learn/step/4425 "Kotlin에서 문자열 템플릿은 문자열을 포맷하는 강력한 도구로, 변수나 표현식 값을 문자열에 직접 삽입할 수 있도록 합니다. | 동적 문자열을 생성하는 편리한 방법으로, 수동으로 문자열을 연결할 필요가 없습니다. 문자열 템플릿은 큰따옴표를 사용하여 정의할 수 있으며, 변수나 표현식의 값은 $ 연산자를 사용하여 삽입할 수 있습니다. 예를 들어, 문자열에 변수 값을 추가하려면 변수 이름 앞에 달러 기호를 써서 코드를 더 읽기 쉽게 만들 수 있습니다. 다음 두 코드 조각은 모두 같은 메시지를 출력합니다. ``` println(Hello, $name!) println(Hello, + name + !) ``` 문자열을 출력하지 않으려면 대신 변수를 생성할 수 있습니다. 문자열 연결 대신 문자열 템플릿을 사용하여 변수 값으로 문자열을 작성하는 것이 좋습니다.") [표현식](https://hyperskill.org/learn/step/4425 "코틀린에서 표현식은 값으로 평가되는 코드 조각입니다. | 표현식은 간단한 리터럴 값, 변수, 함수 호출 또는 연산자나 다른 언어 구문을 사용하여 이러한 요소들을 더 복잡하게 조합한 것일 수 있습니다. 표현식은 함수에서 값을 계산하고 반환하고, 변수에 값을 할당하고, 조건문에서 계산이나 연산을 수행하는 데 사용됩니다. 예를 들어, 2 \* 2는 표현식이며 단일 값을 생성합니다.") 의 값을 문자열에 직접 삽입하는 데 사용됩니다.

사용 예:

변수 값 삽입:

```kotlin
val name = "Alice"
println("Hello, $name!") // Output: Hello, Alice!
```

표현식의 값 삽입:

```kotlin
val a = 5
val b = 10
println("Sum of $a and $b is ${a + b}") // Output: Sum of 5 and 10 is 15
```

좀 더 복잡한 표현식을 삽입하거나 개체 속성에 액세스하려는 경우에는 `{}`표현식을 중괄호로 묶습니다.

음, Kotlin으로 인쇄하는 방법은 짐작하셨겠지만, `$`몇 가지 방법을 알려드리겠습니다.

```kotlin
val a = 20

// There is a dot after $, not variable, so no problem here
println("The price is $a$.") // Output:  The price is 20$. 

// The second $ insert the variable and the first a common symbol here
println("The price is $$a.") // Output: The price is $20. 

// \$ means that you take $ and don't mind about special interpretation
println("The price is \$a.") // Output: The price is $a.

//Another example with \$
println("The price is \$$a.") // Output: The price is $20.
```

따라서 Kotlin의 연산자를 사용하면 변수 값과 표현식 결과를 문자열 [리터럴](https://hyperskill.org/learn/step/4425 "In Kotlin, a literal is a value that is directly represented in the code, such as a number, character, or string. | For example, 3, 'a', and hello are all literals in Kotlin. Integer literals, such as 3, can be represented by the type int (32-bit) or long (64-bit) and can be used in arithmetic operations. Characters, such as 'a', are single-character literals that use less memory than strings. Strings, such as hello, are sequences of characters that can be used for handling text.")`$` 에 편리하고 빠르게 삽입할 수 있습니다 .[](https://hyperskill.org/learn/step/4425 "코틀린에서 리터럴은 숫자, 문자 또는 문자열과 같이 코드에서 직접 표현되는 값입니다. | 예를 들어, 3, 'a', hello는 모두 코틀린의 리터럴입니다. 3과 같은 정수 리터럴은 int(32비트) 또는 long(64비트) 유형으로 표현될 수 있으며 산술 연산에 사용될 수 있습니다. 'a'와 같은 문자는 문자열보다 메모리를 덜 사용하는 단일 문자 리터럴입니다. hello와 같은 문자열은 텍스트를 처리하는 데 사용할 수 있는 일련의 문자입니다.")

## 결론

이 주제에서는 정보를 표준 출력으로 리디렉션하는 두 가지 기본 함수인 . `println`와 `print`. `println`는 정보를 출력한 후 줄바꿈을 하고, while 은 `print`정보를 차례로 출력합니다. 이 함수들을 사용하면 문자열뿐만 아니라 다른 문자도 출력할 수 있다는 점을 기억하세요.
