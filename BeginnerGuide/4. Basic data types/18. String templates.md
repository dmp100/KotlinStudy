간단한 해결책입니다. 하지만 완벽할까요? 아닙니다. 좀 어색하죠.

코틀린은 문자열 템플릿을 사용하여 동일한 작업을 더 편리하게 수행할 수 있는 방법을 제공합니다. 어떻게 작동할까요? 문자열에 변수 값을 추가하려면 `$`변수 이름 앞에 달러 기호를 붙입니다.

```kotlin
val city = "Paris"
val temp = "24"

println("The temperature in $city is $temp degrees Celsius.")
```

코드가 더 읽기 쉬워졌습니다. 두 코드 조각 모두 같은 메시지를 출력합니다.

```no-highlight
The temperature in Paris is 24 degrees Celsius.
```

문자열을 출력하고 싶지 않다면 다음과 같이 변수를 만들 수 있습니다.

```kotlin
val value = "55"
val currency = "dollars"
val price = "$value $currency" // "55 dollars"
```

변수 값을 사용하여 문자열을 작성하려면 문자열 템플릿을 사용하는 것이 좋습니다. 문자열 연결 대신 사용해 보세요.

## 표현식을 위한 템플릿

문자열 템플릿을 사용하면 임의의 표현식 결과를 문자열에 넣을 수 있습니다. 이렇게 하려면 `{...}`달러 기호 뒤에 전체 표현식을 중괄호로 묶어야 합니다 `$`. 예를 들면 다음과 같습니다.

```kotlin
val language = "Kotlin"
println("$language has ${language.length} letters in the name")
```

다음이 인쇄됩니다:

```no-highlight
Kotlin has 6 letters in the name
```

다음은 평가될 [표현식](https://hyperskill.org/learn/step/4547 "In Kotlin, an expression is a piece of code that evaluates to a value. | It can be a simple literal value, a variable, a function call, or a more complex combination of these elements using operators or other language constructs. Expressions are used to compute and return values from functions, assign values to variables, and perform calculations or operations in conditionals. For example, 2 \* 2 is an expression, and it produces a single value.")`{language.length}` 입니다 . 중괄호를 생략하면 다른 결과가 출력됩니다.[](https://hyperskill.org/learn/step/4547 "코틀린에서 표현식은 값으로 평가되는 코드 조각입니다. | 표현식은 간단한 리터럴 값, 변수, 함수 호출 또는 연산자나 다른 언어 구문을 사용하여 이러한 요소들을 더 복잡하게 조합한 것일 수 있습니다. 표현식은 함수에서 값을 계산하고 반환하고, 변수에 값을 할당하고, 조건문에서 계산이나 연산을 수행하는 데 사용됩니다. 예를 들어, 2 \* 2는 표현식이며 단일 값을 생성합니다.")

```no-highlight
Kotlin has Kotlin.length letters in the name
```

따라서 이러한 실수를 방지하려면 문자열 템플릿의 표현식에 항상 중괄호를 사용하세요. 변수 값만 출력하려는 경우에는 중괄호를 사용하지 마세요. 중괄호가 제대로 작동하더라도 변수 값만 출력하려는 경우에는 중괄호를 사용하지 마세요.

## 관용구

관용구는 명확하고 읽기 쉬운 코드를 위한 템플릿입니다. 이러한 패턴은 다른 사람들이 코드를 명확하게 이해할 수 있도록 도와주므로 사용하는 것이 좋습니다. 모든 관용구는 커뮤니티에서 권장하므로, 관용구를 사용하면 진정한 코틀린 스타일 코드에 더 가까워질 수 있습니다. 관용구의 전체 목록은 [코틀린 문서](https://kotlinlang.org/docs/idioms.html) 에서 확인할 수 있습니다 .

우리가 살펴볼 관용구는 [문자열 보간](https://kotlinlang.org/docs/idioms.html#string-interpolation) 입니다 .

```kotlin
val language = "Kotlin"
println("Have a nice $language!")        // nice code
println("Have a nice " + language + "!") // bad practice
```
