기억하시겠지만, 이전에 함수에 대해 이야기했습니다. 그런데 함수란 무엇일까요? 어떻게 사용할 수 있을까요? 함수는 **일련의 명령어** 로 , 프로그램에서 함수의 이름을 호출하여 호출할 수 있습니다 . 함수는 하위 프로그램을 나타내며, 표준 출력에 데이터를 출력하거나 제곱근을 계산하는 등의 작업을 수행합니다.

아래 그림은 함수의 작동 방식을 보여줍니다. 기본적으로 함수는 입력 인수( **입력 데이터** )를 처리하여 유용한 결과를 생성하거나 아무것도 생성하지 않는 상자일 뿐입니다.

## 함수 인수

함수를 사용할 때는 함수 이름 뒤에 괄호를 붙여 [호출 할 수 있습니다. 함수가 하나 이상의 인수(입력 데이터)를 받는 경우, 인수는 괄호 안에 전달해야 합니다.](https://hyperskill.org/learn/step/4575 "코틀린에서 'invoke'는 함수를 호출하거나 사용하는 행위를 의미합니다. | 함수를 사용하려면 함수 이름 뒤에 괄호를 붙여 호출할 수 있습니다. 함수가 하나 이상의 인수(입력 데이터)를 받는 경우, 괄호 안에 전달해야 합니다. 예를 들어, 함수가 단일 문자열 인수를 받는 경우 `functionName(string)`과 같이 호출할 수 있습니다. 일부 함수는 인수를 받을 뿐만 아니라 변수에 할당할 수 있는 결과를 생성하기도 합니다. 함수 인수와 결과 생성은 코틀린에서 함수를 사용할 때 기본적인 개념입니다.")

아래 예에서는 `println`단일 문자열 인수로 함수를 호출합니다.

```kotlin
val text = "Hello"
println(text)
```

이 함수는 새 줄을 인쇄할 때 인수를 전혀 받지 않을 수도 있습니다.

```kotlin
println()
```

따라서 일반적인 형태로 함수는 다음과 같이 호출될 수 있습니다.

```kotlin
function1() // invokes function1 without an argument
function2(arg1) // invokes function2 while passing an argument
function3(arg1, arg2) // invokes function3 while passing two arguments
// ... and so on
```

`function`함수의 이름은 어디에 있나요?

## 결과 생성

일부 함수는 인수를 받을 뿐만 아니라 결과를 생성(반환)하기도 합니다. 변수에 할당할 수도 있습니다.

```kotlin
val result = function(arg)
```

인수를 받아서 결과를 생성하는 함수는 일반적인 수학 함수처럼 보입니다.

예를 들어, 숫자의 절댓값을 반환하는 수학 함수를 살펴보겠습니다.

```kotlin
val number = -10
val nonNegNumber = Math.abs(number) // it takes -10 and returns 10
```

함수를 사용하는 이점은 아무것도 구현할 필요 없이, 함수를 호출하기만 하면 작동한다는 것입니다.

함수 이름은 `abs`점 기호 뒤에 적습니다. 여러 함수가 그룹화되어 있기 때문에 `Math`, 그중 하나를 호출하려면 그룹 이름을 써야 합니다. 지금 당장은 자세히 설명하지 않겠지만, 예제와 연습 문제에서 이와 비슷한 내용을 볼 수 있다는 점만 기억해 두세요.

모든 함수는 결과를 **반환합니다**`println` . 함수 자체도 마찬가지입니다.

```kotlin
val result = println("text")
println(result) // kotlin.Unit
```

결과는 이라는 특수 값인데 `Unit`, 사실상 를 의미합니다 `no result`. 함수가 아무것도 반환하지 않으면 를 반환한다는 의미입니다 `Unit`. 지금은 이것만 이해하시면 됩니다. C나 Java 같은 다른 언어를 사용하셨다면 를 라고 생각해도 됩니다 `Void`.
