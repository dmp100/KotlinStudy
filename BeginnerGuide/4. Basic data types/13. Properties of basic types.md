정수 유형 범위는 - (2^(n-1))에서 (2^(n-1)-1로 계산할 수 있으며, 여기서 n은 유형의 비트 수입니다. 기본 유형은 상대적인 용어이므로 주어진 유형이 둘 중 하나일 필요는 없습니다. 이 경우 Exception 및 Error와 같은 유형의 하위 유형은 애플리케이션 내부의 예외 이벤트를 처리합니다.")** 의 분류와 속성에 대해 알아봅니다 . 여러분도 이미 알고 있는 기본 타입도 있을 것입니다. 기본 타입은 의미에 따라 여러 그룹으로 나눌 수 있습니다. 같은 그룹에 속한 타입들은 동작 방식이 유사하지만, 크기가 다르고 따라서 표현하는 값의 범위도 다릅니다.

## 숫자

**Kotlin은 정수** 와 **분수에** 대한 여러 가지 유형을 제공합니다 .

**정수** (0, 1, 2, ...)는 다음 네 가지 유형으로 표현됩니다. `Long`, `Int`, `Short`, `Byte`(가장 큰 것부터 가장 작은 것까지). 이러한 유형은 크기가 다르며 서로 다른 값 범위를 나타낼 수 있습니다. [정수형 범위는 −(2](https://hyperskill.org/learn/step/4455 "Kotlin에서 정수 타입은 소수점 없이 정수를 나타내는 데이터 타입입니다. 32비트에서는 `int`, 64비트에서는 `long` 타입으로 표현할 수 있습니다. 정수 변수를 사용하면 덧셈, 뺄셈, 곱셈, 나눗셈, 모듈러스와 같은 산술 연산을 수행할 수 있습니다. `int` 타입의 변수를 선언하려면 다음 방법 중 하나를 사용할 수 있습니다. `int` 범위 내의 값을 할당하거나, 변수 이름 뒤에 타입을 지정하거나, 값에 `u`(unsigned) 또는 `L`(long) 접미사를 붙여 값을 지정합니다.") n−1 )에서 (2 n−1 )−1까지 계산할 수 있으며 , 여기서 **n** 은 비트 수입니다. 범위에는 0이 포함되므로 상한에서 1을 뺍니다.

- `Byte`: 8비트(1바이트), 범위는 -128~127입니다.
- `Short`: 16비트(2바이트), 범위는 -32768~32767입니다.
- `Int`: 32비트(4바이트), 범위는 −(2 31 )에서 (2 31 )−1까지입니다.
- `Long`: 64비트(8바이트), 범위는 −(2 63 )에서 (2 63 )−1까지입니다.

크기는 변경할 수 없습니다. 운영 체제나 하드웨어에 영향을 받지 않습니다.

가장 일반적인 정수 유형은 `Int`과 입니다 `Long`. 실제로는 을 고수하는 것이 좋습니다 `Int`. 숫자를 더 자유롭게 표현하고 싶다면 을 사용하세요 `Long`.

```kotlin
val zero = 0 // Int
val one = 1  // Int
val oneMillion = 1_000_000  // Int

val twoMillion = 2_000_000L           // Long because it is tagged with L
val bigNumber = 1_000_000_000_000_000 // Long, Kotlin automatically chooses it (Int is too small)
val ten: Long = 10                    // Long because the type is specified

val shortNumber: Short = 15 // Short because the type is specified
val byteNumber: Byte = 15   // Byte because the type is specified
```

**[부동 소수점 타입은](https://hyperskill.org/learn/step/4455 "Kotlin에서 부동 소수점 유형은 소수 부분이 있는 실수를 나타내는 데이터 유형입니다. Kotlin에는 `Float`(32비트)와 `Double`(64비트)의 두 가지 유형이 있습니다. 이러한 유형은 제한된 수의 십진수(`Float`의 경우 약 6-7개, `Double`의 경우 약 14-16개)를 저장할 수 있습니다. 부동 소수점 유형은 십진수 값을 포함하는 수학 연산에 필수적입니다. 부동 소수점 숫자가 이진수로 표현되는 방식으로 인해 특정 산술 연산에서 계산 중 오류가 발생할 수 있다는 점에 유의해야 합니다. `Float` 및 `Double`을 포함한 숫자 유형의 최댓값과 최솟값을 표시하려면 유형 이름 뒤에 점을 쓰고 `MIN_VALUE` 또는 `MAX_VALUE`를 입력합니다.")** 소수 부분을 포함하는 숫자를 나타냅니다. Kotlin에는`Double`(64비트)와, 32비트의 경우 약 14~16자리`Float`만 저장할 수 있습니다. 실제로는 이타입이 더 일반적입니다.`Float``Double``Double`

```kotlin
val pi = 3.1415              // Double
val e = 2.71828f             // Float because it is tagged with f
val fraction: Float = 1.51f  // Float because the type is specified
```

`Double`숫자형( 및 포함)의 최대값과 최소값을 표시하려면 `Float`유형 이름을 쓰고 그 뒤에 마침표를 찍은 `.`후 `MIN_VALUE`또는 을 사용합니다 `MAX_VALUE`.

```kotlin
println(Int.MIN_VALUE)  // -2147483648
println(Int.MAX_VALUE)  // 2147483647
println(Long.MIN_VALUE) // -9223372036854775808
println(Long.MAX_VALUE) // 9223372036854775807
```

정수형의 크기를 바이트 또는 비트 단위로 가져오는 것도 가능합니다(1바이트 = 8비트):

```kotlin
println(Int.SIZE_BYTES) // 4
println(Int.SIZE_BITS)  // 32
```
