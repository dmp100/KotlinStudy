`c`정수 변수가 보다 크거나 같은지 `a`, 보다 작거나 같은지 확인해야 하는 상황을 상상해 보세요 `b`. 이를 위해 다음과 같이 작성할 수 있습니다.

```kotlin
val within = a <= c && c <= b
```

**이 코드는 잘 작동합니다. 하지만 Kotlin은 범위를** 사용하여 동일한 작업을 수행하는 더 편리한 방법을 제공합니다 .

```kotlin
val within = c in a..b
```

여기서 는 에서 까지 `a..b`의 숫자 범위 (두 경계 값 모두 포함)이고, 는 값이 범위 내에 있는지 확인하는 데 사용되는 **특수 키워드** 입니다 . 나중에 이 키워드가 다른 유형과도 함께 사용될 수 있음을 알게 될 것입니다. **이는 닫힌 범위입니다** .`a``b``in`

또한, 우리는 **개방형** 범위를 갖습니다. 는 부터 까지 `a..<b`의 숫자 범위입니다 (경계 값 제외 ).`a``b``b`

범위에 포함되는 경우 값은 `within`이고 , 그렇지 않은 경우 값은 입니다 .`true``c``false`

다음은 몇 가지 예입니다.

```kotlin
println(5 in 5..15)  // true
println(12 in 5..15) // true
println(15 in 5..15) // true
println(20 in 5..15) // false
println(5 in 5..<15)  // true
println(15 in 5..<15) // false
```

오른쪽 테두리를 제외해야 하는 경우 테두리에서 1을 빼거나 를 사용하여 [개방형 범위를](https://hyperskill.org/learn/step/4633 "In Kotlin, an open-ended range is a range of numbers that goes from a starting value to a finishing value, excluding the finishing value. | The value of a variable is considered to be in the range inclusively if it is greater than or equal to the starting value and less than the finishing value. If the variable's value does not fall within the range, it will be false. To exclude the right border, you can either subtract 1 from it or use the 'until' keyword to create an open-ended range, which is the recommended way.")`..<` 구할 수 있습니다 (권장되는 방법).[](https://hyperskill.org/learn/step/4633 "코틀린에서 열린 범위는 시작 값부터 마지막 ​​값까지의 숫자 범위이며, 마지막 값은 제외됩니다. | 변수의 값은 시작 값보다 크거나 같고 마지막 값보다 작을 경우(포함) 범위에 있는 것으로 간주됩니다. 변수 값이 범위에 포함되지 않으면 false가 됩니다. 오른쪽 테두리를 제외하려면 해당 테두리에서 1을 빼거나 'until' 키워드를 사용하여 열린 범위를 만들 수 있습니다. 이 방법이 권장됩니다.")

```kotlin
val withinExclRight = c in a..b - 1 // a <= c && c < b
val withinExclRight = c in a..<b // a <= c && c < b (the recommended way)
```

값이 범위 내에 없는지 확인해야 하는 경우 `!`앞에 (not)을 추가하면 됩니다 `in`.

```kotlin
val notWithin = 100 !in 10..99 // true
```

표준 논리 연산자를 사용하여 범위를 결합할 수 있습니다. 아래 코드는 가 `c`세 범위 중 하나에 속하는지 확인합니다.

```kotlin
val within = c in 5..10 || c in 20..30 || c in 40..50 // true if c is within at least one range
```

변수에 범위를 할당하여 나중에 사용할 수 있습니다.

```kotlin
val range = 100..200
println(300 in range) // false
```

정수 범위 외에도 문자 범위와 문자열 범위(사전 순서에 따라)도 사용할 수 있습니다.

```kotlin
println('b' in 'a'..'c') // true
println('k' in 'a'..'e') // false

println("hello" in "he".."hi") // true
println("abc" in "aab".."aac") // false
```

정수와 문자의 범위를 이해하는 데는 이 정도면 충분합니다. 다른 유형의 범위는 여기서 다루지 않겠습니다.
