## 루프 반복

가장 간단한 루프를 사용하려면 `repeat(n)`중괄호로 묶인 일련의 명령문을 작성합니다 `{...}`. `n`는 이러한 명령문이 반복되는 횟수를 결정하는 정수입니다.

```kotlin
repeat(n) {
    // statements
}
```

예를 들어, 아래 프로그램은 `Hello`세 번 출력합니다.

```kotlin
fun main() {
    repeat(3) {
        println("Hello")
    }
}
```

산출:

```kotlin
Hello
Hello
Hello
```

**팁:**`n` 가 0이거나 음수 이면 Kotlin은 루프를 무시합니다. `n`가 1이면 명령문은 한 번만 실행됩니다.

Kotlin은 다음 이름으로 현재 반복을 제어할 수 있는 기회를 갖습니다 `it`.

```kotlin
fun main() {
    repeat(3) {
        println(it)
    }
}
```

산출:

```kotlin
0
1
2
```

[중괄호](https://hyperskill.org/learn/step/4643 "코틀린에서 중괄호 `{}`는 0개 이상의 명령문을 포함할 수 있는 코드 블록을 나타내는 데 사용됩니다. | 또한 문자열 템플릿에서 복잡한 표현식이나 객체 속성을 평가하는 데에도 중괄호를 사용합니다. 표현식이 더 복잡한 경우, 달러 기호 `$` 다음에 중괄호를 사용하여 표현식을 감싸고 사용할 수 있습니다. 문자열 템플릿에서는 실수를 방지하고 코드 가독성을 높이기 위해 항상 표현식에 중괄호를 사용하는 것이 좋습니다.") 안의 코드는 일반적으로 " [람다 표현식](https://hyperskill.org/learn/step/4643 "코틀린에서 람다 표현식은 이름이 없는 함수로, 코드를 데이터로 사용하고 전달할 수 있도록 합니다. | 람다 표현식은 함수 참조보다 가독성이 뛰어나고, 개발자가 프로그램을 명확한 역할을 가진 더 작은 함수로 나누어야 합니다. 예를 들어, 함수를 직접 호출하거나 함수 참조를 사용하여 값을 전달할 수 있습니다. 람다 표현식은 최상위 함수처럼 적절한 타입을 가집니다. 람다 표현식에 인수가 없으면 화살표 기호를 포함할 필요가 없습니다. 이름이 없는 함수를 사용하는 방법이 궁금할 수도 있습니다.") " 이라고 합니다 . 이름이 없고 표현식으로 즉시 전달되는 함수입니다. 람다에 대한 자세한 내용은 [여기 에서](https://hyperskill.org/learn/step/30148) 확인할 수 있습니다 .

## 루프에서 데이터 읽기 및 처리

표준 입력에서 데이터를 읽고, 변수를 선언하고, 심지어 `repeat`문장 내에서 계산을 수행할 수도 있습니다.

다음 예제를 살펴보세요. 아래 프로그램은 표준 입력에서 숫자를 읽어서 합을 계산합니다. 첫 번째 숫자는 합에 포함되지 않지만, 입력 시퀀스의 길이를 결정합니다.

```kotlin
fun main() {    
    val n = readln().toInt()
    var sum = 0
    
    repeat(n) {
        val next = readln().toInt()
        sum += next
    }
    
    println(sum)
}
```

이 코드는 숫자 시퀀스의 길이를 읽어 `n`변수에 할당합니다. 그런 다음 총합을 저장하는 변수를 생성합니다. 다음 숫자를 읽어서 정확히 `n`몇 번이나 더하는지 계산합니다. 그런 다음 루프가 멈추고 프로그램은 총합을 출력합니다.

입력 숫자의 예는 다음과 같습니다.

```kotlin
5
40
15
30
25
50
```

산출:

```kotlin
160
```

## 결론

루프는 코드에서 특정 문장을 반복해야 할 때 유용합니다. 이 주제에서는 `repeat()`루프에 대해 살펴보았습니다. 루프는 매우 간단한 도구입니다. 실행 횟수를 정의하고, 중괄호 안에 문장을 넣으면 됩니다! 다음 레슨에서는 고급 루프에 대해 알아보고, 더 복잡한 문제에 루프를 적용하는 방법을 알아보겠습니다.
