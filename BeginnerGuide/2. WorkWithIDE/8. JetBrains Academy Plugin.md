교육용 제품을 사용하면 JetBrains IntelliJ 플랫폼 기반 IDE 내에서 코딩 과제 및 맞춤형 검증 테스트 형태로 프로그래밍 언어를 배우고 가르칠 수 있습니다. JetBrains Academy 플러그인은 다음 링크에서 다운로드하여 설치할 수 있습니다. [https://plugins.jetbrains.com/plugin/10081-jetbrains-academy](https://plugins.jetbrains.com/plugin/10081-jetbrains-academy)

코스를 열면 프로젝트 보기, 작업 설명, 편집기 등 주요 도구 창이 표시됩니다.

## 프로젝트 보기  
  
![](https://ucarecdn.com/48d0cf1c-70af-4675-ba18-2c175684404d/-/crop/830x243/0,4/-/preview/)

프로젝트 보기는 수업 계획서, 즉 수업 단계 또는 과제가 포함된 수업 목록을 보여줍니다. 과제로 이동하려면 해당 과제 이름을 클릭하세요.

**참고:** 프로젝트 도구 버튼을 클릭하거나 `⌘ 1`/ `Alt + 1`단축키를 사용하여 프로젝트 뷰 창을 숨길 수 있습니다. 이렇게 하면 코드와 작업 설명을 위한 공간이 더 넓어집니다. 프로젝트 도구 버튼을 다시 클릭하거나 단축키를 실행하면 숨겨진 프로젝트 뷰가 표시됩니다.

## 작업 설명  
![](https://ucarecdn.com/a873ffcb-8b21-481c-a238-b55ecdac8d67/)

작업 설명 창은 프로그래밍 문제나 프로젝트 단계를 완료하는 데 필요한 모든 정보를 제공합니다.

다음 작업에 대해 작업 설명 아이콘을 사용하세요.

![](https://ucarecdn.com/773f9dd2-71e5-4e8c-be86-299f16f4423c/)퀴즈와 프로그래밍 작업에 대한 작업 할당이 올바르게 수행되었는지 확인하기 위해

![](https://ucarecdn.com/aae24ccc-17c3-464e-9f36-ed5de27cb065/)이전/다음 작업으로 이동하다;

![](https://ucarecdn.com/6621cbca-76b5-4d6b-887f-9a10a74a35f5/)모든 작업 변경 사항을 삭제하고 새로 시작합니다.

![JetBrains IDE: 실행 버튼](https://ucarecdn.com/7fc77294-b534-4e7e-a584-0e9e2c5456eb/)편집기 왼쪽에 있는 실행 버튼을 사용하여 코드를 실행합니다. 조합 `⌃ ⌥ R`/을 `Alt + Shift + F10`사용하면 실행 메뉴를 사용하여 실행할 항목을 선택할 수 있습니다.

![](https://ucarecdn.com/89844900-8b9c-47b3-a3bd-c7818267b9bc/)Hyperskill의 작업 페이지를 보고 댓글을 남겨보세요.

작업 설명은 눈에 띄어야 하며, 완전히 숨기는 것은 권장하지 않습니다. 하지만 눈에 띄지 않는 곳으로 옮기면 방해를 덜 받을 수 있습니다.

두 개의 모니터로 작업하는 경우 가장 좋은 방법 중 하나는 작업 설명 패널을 플로팅 모드로 전환하여 두 번째 모니터로 옮기거나 기본 IDE 창 근처에 두는 것입니다.

![](https://ucarecdn.com/6c180dbb-bf91-4c30-8005-93808c7b21cc/) 특수 도구 창 설정 아이콘을 사용하면 이 작업을 수행할 수 있습니다.

![](https://ucarecdn.com/af0e4eb0-aa77-4ef4-9f55-79006d8c57e7/-/preview/)

원하시면 패널을 왼쪽이나 아래쪽으로 이동할 수 있습니다.  
![](https://ucarecdn.com/7ade7de4-bf34-4925-ae58-1ec6ea496516/)

## 편집자  
![](https://ucarecdn.com/3bf9a8f4-c44d-435c-b338-a07f5a152b05/)

편집기는 주로 프로그래밍을 하는 공간입니다. 이론 과제와 퀴즈를 위해 별도의 확인 없이 자유롭게 실험해 볼 수 있습니다. 편집기에서 프로그래밍 과제를 수행하려면 수정해야 할 코드를 찾거나 직접 코드를 작성해야 합니다. 이 코드는 확인될 것입니다.

![JetBrains IDE: 실행 아이콘](https://ucarecdn.com/821767a3-9046-47e2-a541-8da61076fe4f/)왼쪽에 있는 이 아이콘을 사용하면 언제든지 코드를 실행할 수 있습니다.

**참고:** 편집기로 빠르게 돌아가 코드에 집중하고 싶다면 모든 창 숨기기 명령( `⌘ ⇧ F12`/ `Ctrl + Shift + F12`)을 사용하는 것이 가장 좋습니다. 이 명령을 다시 한 번 실행하면 모든 창이 다시 나타납니다.

## 결론

보시다시피, IntelliJ 플랫폼 기반 IDE에는 학습 과정을 간소화하는 데 도움이 되는 유용한 도구들이 포함되어 있습니다. 이러한 도구를 사용하면 학습 과정을 최적화하고, 시간을 절약하고, 작업 환경에 더 익숙해질 수 있습니다.
