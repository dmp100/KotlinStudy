이 주제에서는 사용자 입력에서 정보를 읽고 상호작용하는 방법을 알아봅니다. 프로그램은 사용자에게 특정 정보를 요구하는 경우가 많습니다. 또한 함수를 사용하여 변수를 관리하고 `readln()`다양한 데이터 유형을 처리하는 방법도 알아봅니다.

## 표준 입력

**표준 입력은** 프로그램으로 들어오는 데이터 스트림입니다. 운영 체제에서 지원됩니다. 기본적으로 표준 입력은 키보드에서 데이터를 받지만, 파일에서도 데이터를 받을 수 있습니다.

모든 프로그램이 표준 입력을 사용할 필요는 없지만, 꽤 자주 사용하게 될 가능성이 높습니다. 프로그래밍 문제를 해결하는 일반적인 방법은 다음과 같습니다.

- 표준 입력에서 데이터를 읽습니다.
    
- 결과를 얻기 위해 데이터를 처리합니다.
    
- 결과를 표준 출력으로 출력합니다.
    

유용한 기능을 수행하는 프로그램을 작성하기 전에 표준 입력에서 데이터를 읽는 방법을 이해해야 합니다.

## readln 사용하기

코틀린에서는 표준 입력에서 데이터를 읽으려면 함수를 사용할 수 있습니다 `readln()`. 이 함수는 전체 줄을 문자열로 읽습니다.

```kotlin
val line = readln()
```

변수 line은 String 유형을 갖는데, 그 이유는 [표현식이](https://hyperskill.org/learn/step/11224 "코틀린에서 표현식은 값으로 평가되는 코드 조각입니다. | 표현식은 간단한 리터럴 값, 변수, 함수 호출 또는 연산자나 다른 언어 구문을 사용하여 이러한 요소들을 더 복잡하게 조합한 것일 수 있습니다. 표현식은 함수에서 값을 계산하고 반환하고, 변수에 값을 할당하고, 조건문에서 계산이나 연산을 수행하는 데 사용됩니다. 예를 들어, 2 \* 2는 표현식이며 단일 값을 생성합니다.") `readln()` 이 유형의 값을 반환하기 때문입니다.

`readLine()!!`이전 버전의 Kotlin 언어를 사용하는 경우 . 대신 . 를 사용해야 합니다 `readln()`. 이 함수들은 동일한 기능을 수행하지만 `readln()`더 짧고 선호되므로 Kotlin 1.6 이상을 사용하는 경우 . 를 사용하는 것이 좋습니다 `readln()`.

```kotlin
val line = readLine()!! // before Kotlin 1.6
```

느낌표에 대해 조금 헷갈리실 수도 있습니다. 이 구문은 컴파일러에게 비어 있지 않은 입력을 보장합니다. 나중에 더 자세히 설명하겠습니다.

다음은 표준 입력에서 한 줄을 읽어 표준 출력으로 보내는 간단한 프로그램입니다.

```kotlin
fun main() {
    val line = readln()
    println(line)
}
```

유효한 입력 데이터의 예는 다음과 같습니다.

```no-highlight
Hello, Kotlin
```

출력 결과는 다음과 같습니다.

```no-highlight
Hello, Kotlin
```

_기본적으로, 입력한 데이터에 대한 정보를 보려면 Enter 키를_ 눌러야 합니다 .

이제 .를 사용하여 다양한 유형의 데이터를 읽는 방법을 살펴보겠습니다 `readln()`.

## toInt() 및 toLong()

때로는 사용자로부터 숫자형 데이터를 얻어야 합니다. 예를 들어 사용자의 나이 또는 졸업 연도에 대한 정보를 얻어야 한다고 가정해 보겠습니다. 숫자형 값을 사용하려면 함수를 사용합니다 `toInt()`. 입력값을 정수로 변환하려면 `toInt()`함수와 점(.) 구문을 사용합니다.

```kotlin
println("Enter any number: ") 
val number = readln().toInt() 
print("You entered the number: ")
print(number) 
```

유효한 입력 데이터의 예는 다음과 같습니다.

```no-highlight
56
```

출력은 다음과 같습니다.

```no-highlight
Enter any number: 
56 
You entered the number: 56
```

예를 들어, 크고 호화로운 요트의 비용과 같이 더 큰 숫자를 처리해야 하는 경우 다음 함수를 사용하세요 `toLong()`.

```kotlin
println("How much is your yacht worth?")
val cost = readln().toLong()
print("You entered: ")
print(cost)
```

출력은 다음과 같습니다.

```no-highlight
How much is your yacht worth?
10000000000
You entered: 10000000000
```

## toDouble() 및 toBoolean()

더 정확한 값을 구해야 한다면 어떻게 해야 할까요? 예를 들어 휘발유 1리터의 정확한 가격을 알아야 한다고 가정해 보겠습니다. 이 경우에는 정수가 없을 가능성이 높기 때문에 `toInt()`또는 를 사용할 수 없습니다. 이 경우 다음 함수를 사용합니다 .`toLong()``toDouble()`

```kotlin
println("Enter any double type number:")
val number = readln().toDouble()
println("You entered the number: ")
print(number)
```

출력은 다음과 같습니다.

```no-highlight
Enter any double type number:
0.5673421
You entered the number: 
0.5673421
```

동일한 논리가 Boolean에도 적용됩니다. 다음 함수를 사용하세요 `toBoolean()`.

```kotlin
println("The earth is flat. Type true or false:")
val answer = readln().toBoolean()
print("The earth is flat: ")
print(answer)
```

출력은 다음과 같습니다.

```no-highlight
The earth is flat. Type true or false:
false
The earth is flat: false
```

## 다중 입력

여러 입력을 받고 처리할 수 있을까요? 정답은 '예'입니다. 함수 값을 갖는 여러 변수를 선언하기만 하면 됩니다 `readln()`. 여러 값을 입력하려고 하는데 데이터 유형이 서로 다른 경우, _Enter 키를_ 눌러 유형을 구분해야 합니다.

여러 값의 입력과 출력을 보여주는 예를 생각해 보겠습니다.

```kotlin
val a = readln()
val b = readln().toInt()
val c = readln()
print(a)
print(" ")
print(b)
print(" ")
print(c)
```

`String`유효한 값 과 `Int`입력 값 의 예는 다음과 같습니다 .

```no-highlight
You earned
100
points!
```

출력은 다음과 같습니다.

```no-highlight
You earned 100 points!
```

보시다시피, 서로 다른 데이터 유형의 여러 값을 가져오는 것은 어렵지 않습니다. 여러 변수를 선언하고, 원하는 `readln()`함수를 할당하고, 콘솔에 올바르게 표시하기만 하면 됩니다.

## 한 줄에 여러 값 읽기

한 줄에 두 개의 값을 읽어야 하는 경우 다음 구문을 사용할 수 있습니다.

```kotlin
val (a, b) = readln().split(" ")
println(a)
println(b)
```

입력 데이터의 예는 다음과 같습니다.

```no-highlight
Hello, Kotlin
```

출력 결과는 다음과 같습니다.

```no-highlight
Hello,
Kotlin
```

이 구문은 입력 문자열을 공백으로 나누어 각 단어를 변수 `a`와 에 저장합니다 `b`. 이에 대해서는 나중에 더 자세히 살펴보겠습니다.

같은 방식으로 줄당 최대 4개의 값을 읽을 수 있습니다.

```kotlin
val (a, b, c, d) = readln().split(" ")
println(a)
println(b)
println(c)
println(d)
```

입력 데이터의 예는 다음과 같습니다.

```no-highlight
Have a nice Kotlin
```

출력 결과는 다음과 같습니다.

```no-highlight
Have
a
nice
Kotlin
```

## 결론

보시다시피 입력을 받는 것은 매우 간단합니다. 함수의 값을 가진 변수를 선언하기만 하면 됩니다 `readln()`. `readln()`함수는 입력을 자동으로 해당 타입으로 변환합니다 . 프로그램은 타입 변환을 사용하여 문자열, 숫자, 부울 입력을 처리할 수 있습니다. 변환하려면 , , , , `String`함수를 사용합니다 . 또한 동일하거나 다른 타입의 여러 값을 입력하고 출력할 수도 있습니다.`toInt()``toLong()``toDouble()``toBoolean()`
