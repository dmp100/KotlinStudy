문자열은 큰따옴표로 묶인 문자들의 시퀀스입니다. 코틀린에서 문자열은 `String`타입으로 표현됩니다. 타입은 실무에서 가장 널리 사용되는 [데이터 타입](https://hyperskill.org/learn/step/12561 "Kotlin에서 데이터 유형은 변수, 표현식 또는 함수 매개변수가 보유할 수 있는 값의 종류를 지정하는 분류입니다. | Kotlin과 같은 정적 타입 언어의 필수적인 측면으로, 변수와 함수 매개변수에 대한 명시적 또는 암시적 타입 선언이 필요합니다. 예를 들어 Kotlin에는 정수, 부동 소수점 숫자, 문자, 부울, 문자열과 같은 다양한 데이터 유형이 있습니다. 이 중 문자열은 큰따옴표로 묶인 0개 이상의 문자 시퀀스를 나타내는 널리 사용되는 데이터 유형입니다. 문자열은 조건문과 정규 표현식에서 자주 사용되며 길이를 얻거나 두 문자열을 연결하는 등 다양한 연산을 수행합니다. 또한, 타입 추론은 Kotlin의 유용한 기능으로, 컴파일러가 변수, 표현식 또는 함수 매개변수의 데이터 유형을 자동으로 추론하여 코드를 더 간결하고 읽기 쉽게 만듭니다.") 중 하나입니다 . 이 주제에서는 문자열을 다루는 몇 가지 연산을 알아보겠습니다.

## 문자 접근

문자열의 요소는 인덱스로 액세스할 수 있는 개별 문자입니다. 문자열의 첫 번째 요소는 인덱스를 가집니다.00.

```kotlin
val greeting = "Hello"

val first = greeting[0]  // 'H'
val second = greeting[1] // 'e'
val five = greeting[4]   // 'o'
```

마지막 요소의 인덱스는 문자열 길이에서 1을 뺀 값입니다. 문자열 의 경우 `"Hello"`마지막 요소는 입니다 `'o'`. 문자열의 길이가 5이므로 인덱스는 4입니다.

```kotlin
val last = greeting[greeting.length - 1] // 'o'
val prelast = greeting[greeting.length - 2] // 'l'
```

Kotlin은 문자열의 첫 번째 문자와 마지막 문자에 액세스하는 여러 가지 편리한 방법을 제공합니다.

```kotlin
println(greeting.first())   // 'H'
println(greeting.last())    // 'o'
println(greeting.lastIndex) // 4
```

이 기능을 사용하면 더 읽기 쉬운 코드를 작성할 수 있습니다.

## 빈 문자열

때로는 가 비어 있는지 확인하는 빠른 방법이 필요합니다 `String`. 예를 들어 문자열의 첫 번째 요소가 필요하고 그 존재 여부를 확인하고 싶을 때, 길이를 사용하여 0보다 큰지 확인할 수 있습니다. 하지만 훨씬 더 우아한 방법은 다음 함수를 사용하는 것입니다 `isEmpty()`.

```kotlin
val emptyString = ""
println(emptyString.length == 0) //true
println(emptyString.isEmpty()) //true
```

## 불변성

문자열은 **변경 불가능합니다** . 즉, 한 번 생성되면 문자열은 그대로 유지됩니다. 문자열의 요소를 수정할 수 없습니다. 따라서 아래 예제는 작동하지 않습니다.

```kotlin
val valString = "string"
valString[3] = 'o' // an error here!
var varString = "string"
varString[3] = 'o' // an error here too!
```

문자열을 변경해야 하는 경우 다음과 같이 다시 할당할 수 있습니다.

```kotlin
var varString = "string"
varString = "strong" // legal
val valString = "string"
valString = "strong" // error, you cannot reassign val
```

실제로 변수에 저장된 값을 수정하는 것이 아니라 `varString`, 새로운 값을 할당합니다. 따라서 이는 절대적으로 허용됩니다. 이는 문자열을 다루는 방법 중 하나입니다. 문자열을 수정해야 하는 경우 새 문자열을 생성하기만 하면 됩니다.

## 문자열 비교

두 문자열을 비교하려면 `==`(같음) 및 `!=`(같지 않음) 연산자를 사용합니다. 두 연산자 모두 두 문자열을 [피연산자](https://hyperskill.org/learn/step/12561 "코틀린에서 피연산자는 연산자가 적용되는 값 또는 변수입니다. | 예를 들어, 표현식 1 + 3에서 1과 3은 피연산자이고 +는 연산자입니다. 피연산자는 리터럴, 변수 또는 더 복잡한 표현식일 수 있습니다. 코틀린에는 덧셈(+), 뺄셈(-), 곱셈(*), 나눗셈(/), 나머지(%)의 다섯 가지 산술 연산자가 있습니다. 이러한 연산자는 이진 연산자로, 두 개의 피연산자를 사용합니다. 피연산자의 평가는 연산자 우선순위라는 일련의 규칙에 따라 수행되며, 연산자는 이 규칙에 따라 적용됩니다.")`Boolean` 로 사용하여 ( `true`또는 ) 유형 의 값을 반환합니다 `false`.

```kotlin
val first = "first"
val second = "second"
var str = "first"

println(first == str)    // true
println(first == second) // false
println(first != second) // true
```
