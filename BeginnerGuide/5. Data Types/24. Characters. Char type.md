이 주제에서는 Kotlin에서 문자를 다루는 방법을 배웁니다. 이는 매우 기본적인 개념이며, 이 개념에 익숙해지면 텍스트 작업 방법을 더 잘 이해하는 데 도움이 될 것입니다.

## 캐릭터란 무엇인가

[각 문자는 작은따옴표](https://hyperskill.org/learn/step/4680 "In Kotlin, a single quote is used to define a character literal, which represents a single symbol such as an alphabet letter, a digit from 0 to 9, whitespaces, or other symbols (e.g. | <, >). A character literal can be enclosed in single quotes and can represent any symbol including special characters and Unicode characters. It's important to note that a character literal can only represent a single symbol and should not be confused with a number or a string, which are sequences of characters. Additionally, character literals can be represented by their hexadecimal code in the Unicode table.") 로 묶인 기호일 뿐입니다 . `Char`유형은 문자(대문자와 소문자 모두), 숫자 및 기타 기호를 나타냅니다.

```kotlin
val lowerCaseLetter: Char = 'a'
val upperCaseLetter: Char = 'Q'
val number: Char = '1'
val space: Char = ' '
val dollar: Char = '$'
```

이 유형은 상형문자를 포함한 모든 기호는 물론 일부 특수 기호도 표현할 수 있습니다.

문자는 [유니코드 표의](https://unicode-table.com/en/) 16진수 코드로도 표현될 수 있습니다 . 코드는 다음으로 시작합니다 `\u`.

```kotlin
val ch = '\u0040' // it represents '@'
println(ch) // @
```

이러한 코드를 표현하기 위해 일련의 문자를 사용하지만, 코드 자체는 정확히 하나의 문자를 표현합니다.

예를 들어, 라틴 대문자의 16진수 코드는 `'\u0041'`~ `'\u005A'`이고, 라틴 소문자의 16진수 코드는 `'\u0061'`~ 입니다 `'\u007A'`.

숫자를 문자로, 또는 그 반대로 변환할 수도 있습니다. 작동 방식을 살펴보겠습니다.

```kotlin
val ch = 'a'
println(ch.code)   // 97
val num = 97
println(num.toChar()) // a
```

## 문자를 읽는 방법

코틀린은 . 를 읽는 편리한 함수를 제공하지 않습니다 `Char`. 하지만 다른 방법으로도 가능합니다. 한 `Char`줄 전체에서 읽어야 하는 경우 다음 구문을 사용하세요.

```kotlin
val ch: Char = readln().first()
```

방금 a를 읽어서 `String`첫 번째 요소인 a를 얻었습니다 `Char`. 문자열은 문자로 구성되지만, 이에 대해서는 나중에 더 자세히 살펴보겠습니다.

## 이후 문자 검색

유니코드 순서에 따라 다음 문자와 이전 문자를 구하기 위해 정수를 더하고( `+`) 빼는( ) 두 가지 연산자가 있습니다 .`-`

```kotlin
val ch1 = 'b'
val ch2 = ch1 + 1 // 'c'
val ch3 = ch2 - 2 // 'a'
```

숫자에 기호를 추가하면 오류가 발생하므로 기억하세요.

```kotlin
val ch = 'a'
val ch1 = 1 + ch // Error
```

두 문자를 합산할 수도 없습니다.

```kotlin
val charsSum = 'a' + 'b' // Error
```

증가( `++`)와 감소( `--`) 연산자는 접두사 및 접미사 형태로 사용할 수 있습니다. 대입 연산자는 `+`or 와 결합하여 `-`문자에 사용할 수 있으며, `+=`and 에도 사용할 수 있습니다 `-=`.

```kotlin
var ch = 'A'

ch += 10
println(ch)   // 'K'
println(++ch) // 'L'
println(++ch) // 'M'
println(--ch) // 'L'
```

문자를 숫자로 곱하거나 나눌 수 없습니다.

## 이스케이프 시퀀스

백슬래시로 시작하는 특수 문자 중에는 **이스케이프** 또는 **제어 시퀀스**`\` 라고 하는 문자들이 있습니다 . 대부분의 문자에는 해당 기호가 없으며 키보드에서 찾을 수 없습니다. 이러한 문자를 나타내기 위해 일반 기호 쌍을 사용합니다. 프로그램에서 이 쌍은 적절한 코드를 가진 정확히 하나의 문자로 간주됩니다.

- `'\n'`줄바꿈 문자입니다.
    
- `'\t'`탭 문자입니다.
    
- `'\r'`캐리지 리턴 문자입니다.
    
- `'\\'`백슬래시 문자 자체입니다.
    
- `'\''`는 작은따옴표입니다.
    
- `'\"'`[는 큰](https://hyperskill.org/learn/step/4680 "코틀린에서 큰따옴표는 일련의 문자를 나타내는 문자열 유형을 정의하는 데 사용되는 기호입니다. | 코틀린에서 가장 널리 사용되는 데이터 유형 중 하나이며, 큰따옴표로 묶인 문자 수에 따라 결정됩니다. 문자열 연결은 문자열에서 흔히 사용되는 연산으로, 다른 문자열들을 사용하여 하나의 문자열을 구성하는 데 사용됩니다. 두 문자열을 연결하면 새로운 문자열이 생성됩니다.") 따옴표 입니다 .
    

다음은 몇 가지 예입니다.

```java
print('\t') // makes a tab
print('a')  // prints 'a'
print('\n') // goes to a new line
print('c')  // prints 'c'
```

위 코드는 다음과 같이 인쇄됩니다.

```java
  a
c
```

## 문자를 사용한 관계 연산

유니코드 표에서의 위치에 따라 관계 연산( `==`, `<`, `>`, `<=`, `>=`, 및 ) 을 사용하여 문자를 비교할 수 있습니다 .`!=`

```kotlin
println('a' < 'c')  // true
println('x' >= 'z') // false

println('D' == 'D') // true
println('Q' != 'q') // true because capital and lowercase letters are not the same

println('A' < 'a')  // true because capital Latin letters are placed before lowercase ones
```

`Char`관계 연산과 코드를 사용하면 a가 숫자인지 확인할 수 있습니다 . 모든 10개의 숫자는 에서 `'\u0030'`까지 의 코드를 갖습니다 `'\u0039'`.

이를 수행하는 프로그램은 다음과 같습니다.

```kotlin
fun main() {
    val ch = readln().first()
    val isDigit = ch >= '\u0030' && ch <= '\u0039'

    println(isDigit)
}
```

입력값이 숫자, `'0'`, `'1'`, `'2'`, ..., `'8'`또는 `'9'`(따옴표 제외)이면 프로그램은 를 출력합니다 `true`. 그렇지 않으면 를 출력합니다 `false`.

## 문자 처리

문자 작업에 유용한 다양한 함수가 있습니다. 문자 코드를 처리하는 대신 이러한 함수를 사용할 수 있습니다.

- `isDigit()``true`주어진 문자가 숫자( `'1'`, , 등)를 나타내는 경우 반환합니다 `'2'`. 그렇지 않으면 `false`;
    
- `isLetter()``true`주어진 문자가 문자를 나타내는지( `'a'`, `'B'`, `'m'`, 등) 를 반환합니다 . 그렇지 않으면 `false`;
    
- `isLetterOrDigit()``true`주어진 문자가 문자 또는 숫자를 나타내는지 여부 를 반환합니다 `false`. 그렇지 않으면,
    
- `isWhitespace()``true`주어진 문자가 공백( `' '`, 또는 `'\t'`, 또는 `'\n'`) 을 나타내는 경우 를 반환합니다 . 그렇지 않으면 `false`;
    
- `isUpperCase()``true`주어진 문자가 대문자이면 반환합니다 . 그렇지 않으면, `false`;
    
- `isLowerCase()``true`주어진 문자가 소문자이면 반환합니다 . 그렇지 않으면, `false`;
    
- `toUpperCase()`주어진 문자의 대문자 형태를 반환합니다(Kotlin 1.5 이전 버전에서는 사용하지 않는 것이 좋습니다).
    
- `uppercaseChar()`주어진 문자의 대문자 형태를 반환합니다(Kotlin 1.5부터).
    
- `uppercase()``String`주어진 문자의 대문자 형태를 반환합니다 (Kotlin 1.5부터).
    
- `toLowerCase()`주어진 문자의 소문자 형태를 반환합니다(Kotlin 1.5 이전 버전에서는 사용하지 않는 것이 좋습니다).
    
- `lowercaseChar()`주어진 문자의 소문자 형태를 반환합니다(Kotlin 1.5부터).
    
- `lowercase()``String`주어진 문자의 소문자 형태 를 반환합니다 (Kotlin 1.5부터).
    

위에 나열된 기능의 몇 가지 예를 살펴보겠습니다.

```kotlin
val one = '1'

val isDigit = one.isDigit()   // true
val isLetter = one.isLetter() // false

val capital = 'A'
val small = 'e'

val isLetterOrDigit = capital.isLetterOrDigit() // true

val isUpperCase = capital.isUpperCase() // true
val isLowerCase = capital.isLowerCase() // false

val capitalEString = small.uppercase() // "E"
val capitalE = small.uppercaseChar()   // 'E'
```
